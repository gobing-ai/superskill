agent skill `writing-great-skills` (source: mattpocock-skills, productivity
collection), captured 2026-07-11 for analysis and extraction into

be referenced by tasks or code. This in-repo copy is the durable source for the
extraction task; treat it as read-only reference material, not as an installed
skill (`disable-model-invocation: true` is preserved from the source).
